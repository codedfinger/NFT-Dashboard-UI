To configure and use this REST API; you need the following technologies

1. A web browser to view the app
2. install node v12 or newer versions

Copy the contents of the `.env.sample` file, create a new `.env` file and add the conntents into it.

open the project in terminal.

run `$npm install` to download all the packages from the package.json file

in your terminal run `$npm start` to start the server on port 3000 or whatever port you set in the env file.
